This is a simple blog I created to showcase ocean activities. I paint sea waves and surfers, which inspired me to build something related to that passion as a web developer. I added a booking form and a contact page to simulate a real-world site. These pages don’t collect or store data, as the website has no backend functionality — it’s purely for demonstration. The main goal of this project is to highlight my skills in Flexbox, CSS Grid, and other front-end styling techniques.
